InheritanceCSXSWindowedApplication mx.core.WindowedApplication

CSXSWindowedApplication is a replacement for the AIR SDK class mx.core.WindowedApplication in the context of CSXS if AIR functionality is used by an extension. This class extends WindowedApplication with the purpose of removing event listeners with NativeApplication and NativeWindow just before unloading the extension. If this class is not used instead of WindowedApplication, then the extension is not properly unloaded.
