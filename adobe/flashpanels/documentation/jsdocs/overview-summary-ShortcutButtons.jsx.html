ShortcutButtons.jsx

Summary

The script contains two functions to add a new document and to close the foremost document in Adobe Photoshop. This script is to be used by a SWF file called ShortcutButtons.swf created through Adobe Flex.

Author: John Huan Vu, Photoshop Engineering Intern, Adobe Systems Incorporated

/**
	Create a new document within Adobe Photoshop.
*/
function addDocument(){
	app.documents.add();
}

/**
	Close the active document within Adobe Photoshop.
*/
function closeDocument(){
	app.activeDocument.close();
}
