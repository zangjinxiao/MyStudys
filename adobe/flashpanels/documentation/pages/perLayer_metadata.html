Per Layer Metadata Panel: View Metadata

The first part of the Per Layer Metadata Panel is viewing metadata. Metadata are contained within images which can be retrieved and accessed through File Info or applications like Adobe Bridge. Some metadata are editable like the International Press Telecommunications Council (IPTC) metadata while some metadata are not editable like the Exchangeable Image File Format (EXIF) used as camera data. The result of this section is a basic understanding of retrieving and accessing metadata within Adobe Photoshop using a sample image included with the Adobe Photoshop Panel Developer's Guide. For more information about accessing metadata, please refer to Adobe Developer's Center: Extensible Metadata Platform (XMP).

Instructions:

  1. Open Adobe Photoshop.
  2. Open PerLayerMetadataMiddle.jpg under the samples\sample pictures folder included with the Adobe Photoshop Panel Developer's Guide.
  3. Go to File > File Info....

    The File Info shows the metadata stored within PerLayerMetadataMiddle.jpg image. There are multiple tabs across the top allowing the user to view and edit various metadata. The Description field within the Description tab will be used in the Per Layer Metadata tutorial.
  4. Go to the Raw Data tab.

    Next to the cancel button, the user is able to import and export the raw metadata of the document to a file readable by a text editor. The Raw Data will be used in the Per Layer Metadata tutorial.
  5. Close Adobe Photoshop.
