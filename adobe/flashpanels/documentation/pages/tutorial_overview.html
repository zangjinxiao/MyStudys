Adobe® Photoshop® Panel Developer's Guide

Overview

The following tutorials and samples starts with a standard "Hello World" project and then gradually becomes more and more complex. The tutorials are meant to be straight-forward leaving developers to their own imagination.

Go to the JavaScript API that covers a summary of functions and scripts used in the tutorials.

The following is the outline of the tutorials:

  1. Starting a Flex Project
    The Starting a Flex Project tutorial will show how to start Adobe Flex Builder by removing all projects to prepare developers for every tutorial.

  2. Hello World Panel
    The Hello World Panel tutorial will highlight the first project for any computer programmer - having it say "Hello World."

  3. Placing the CSXS Library
    The Placing and Using the CSXS Library tutorial will highlight how to obtain the CSXS Library file from the libraries folder and place in the project's libs folder found in an Adobe Flex project.

  4. Shortcut Buttons Panel
    The Shortcut Buttons Panel tutorial will highlight the direct communication from a SWF file to Adobe Photoshop through a JavaScript file.

  5. Setting Up Script Listener
    The Setting Up Script Listener tutorial will highlight finding the character ID code associated with events in Adobe Photoshop.

  6. Color Picker Panel
    The Color Picker Panel tutorial will highlight the bi-directional communication of a SWF file and Adobe Photoshop through a JavaScript file.

  7. Flickr Search Panel
    Primarily stemming from Adobe Flex Builder 3: Creating a Simple RIA (Rich Interactive Application) Tutorial [Video Tutorial]
    The Flickr Search Panel tutorial will highlight the creation of a service connection, creation of a Adobe Flex module, and changing the Adobe Photoshop Panel's properties.

  8. Per Layer Metadata Panel
    The Per Layer Metadata Panel tutorial will highlight how to access and change the metadata of layers in an Adobe Photoshop document and retrieve the last date and time when a layer has been modified.
