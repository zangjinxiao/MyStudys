Color Picker Panel

The Color Picker Panel tutorial will highlight the bi-directional communication of a SWF file and Adobe Photoshop through a JavaScript file. The process to create the Color Picker Panel is to first program functions through JavaScript, design the Adobe Photoshop Panel using Adobe Flex Builder, finding the character ID code of developer's events, program functions through ActionScript, and creating custom icons for the Adobe Photoshop Panel when it is minimized. The result is an Adobe Photoshop Panel containing a foreground color picker and a background color picker that resembles the same color pickers currently displayed in the Adobe Photoshop tool palette.

The Color Picker Panel tutorial is separated into smaller parts:

  1. JavaScript Programming
  2. Designing the Adobe Photoshop Panel using Adobe Flex Builder
  3. Finding Character ID Code to Register or Unregister Events
  4. ActionScript Programming
  5. Creating Custom Icons
