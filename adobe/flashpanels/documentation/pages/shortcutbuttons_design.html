Shortcut Buttons Panel: Design the Panel

The second part of the Shortcut Buttons Panel is designing the Adobe Photoshop Panel using Adobe Flex Builder. The developer will drag and drop two button control components and two checkbox control components to the design area within Adobe Flex Builder to design the Adobe Photoshop Panel. The result is an Adobe Photoshop Panel containing two buttons and two checkboxes.

Instructions:

  1. Complete the Starting a Flex Project.
  2. Go to File > New > Flex Project.
    1. Under Project name:, type in ShortcutButtons.
      Note: The Adobe Flex Project filename and corresponding SWF filename must have the same name as the JavaScript filename.
    2. Deselect Use default location and under Folder:, save to the Desktop in a new folder called ShortcutButtons.
    3. Press Finish.

  3. Go to Design Mode by selecting Design under the ShortcutButtons.mxml tab or go to Window > Switch Source/Design Mode.
  4. Go to the Components window or go to Window > Components.
    1. Under the Controls folder in the Components window, drag and drop the following control components into the design area:
      • Two Button control components
      • Two CheckBox control components
    2. Double-Click on one of the Button control component placed in the design area and type in New Document and press Enter.
    3. Double-Click on the other Button control component placed in the design area and type in Close Document and press Enter.
    4. Double-Click on one of the CheckBox control component's text placed in the design area and type in Activate every 10 seconds and press Enter.
    5. Double-Click on the other CheckBox control component's text placed in the design area and type in Activate every 12 seconds and press Enter.
  5. Go to File > Save.
  6. Close Adobe Flex Builder.
